# Projeto TecVander - Esquemas e Informações Técnicas

## Visão Geral

Este projeto consiste em um portal web estático e de acesso local (`offline-first`) que centraliza uma vasta gama de informações técnicas, esquemas elétricos e manuais de manutenção para profissionais da área de manutenção de ônibus. O objetivo é fornecer uma ferramenta de consulta rápida e eficiente para eletricistas, mecânicos e técnicos de refrigeração, auxiliando no diagnóstico e reparo de veículos.

A estrutura é baseada em HTML, CSS e JavaScript simples, garantindo compatibilidade com qualquer navegador moderno sem a necessidade de um servidor web ou conexão com a internet para acesso ao conteúdo já baixado.

---

## Conteúdo Disponível

O portal agrega documentação técnica de diversos fabricantes e sistemas, organizada nas seguintes categorias:

- **Mercedes-Benz (MBB):**
  - Chassi Euro 3, Euro 5 e Euro 6.
  - Documentação sobre motorização, arquitetura eletrônica, sistema de pós-tratamento (BlueTec), etc.
- **Marcopolo:**
  - Esquemas elétricos para carrocerias, como Torino.
  - Manuais de treinamento e diagramas de componentes específicos (portas, elevador, validadores).
- **Sistemas de Ar Condicionado:**
  - Manuais e esquemas para equipamentos **Thermo King**, **Spheros**, **Denso** e **Valeo**.
- **SCR / Arla 32:**
  - Informativos técnicos, diagramas e guias de manutenção para o sistema de Redução Catalítica Seletiva.
- **Eletricidade Automotiva:**
  - Uma seção didática com conceitos fundamentais de eletricidade, incluindo calculadoras interativas para Lei de Ohm (Tensão, Corrente, Resistência) e Potência.
- **Vídeos Técnicos:**
  - Uma seleção de vídeos com dicas práticas e tutoriais sobre diagnóstico e manutenção.

---

## Como Utilizar

1.  Clone ou baixe este repositório para sua máquina local.
2.  Abra o arquivo `index.html` em seu navegador de preferência.
3.  Navegue pelas seções para encontrar a documentação desejada. Todos os links para esquemas e apostilas abrirão os arquivos PDF correspondentes.

## Aviso de Direitos Autorais

**Este projeto atua como um agregador de informações técnicas para fins educacionais e de apoio profissional.**

Todo o conteúdo, incluindo esquemas elétricos, manuais de operação, apostilas e diagramas, é de propriedade intelectual de seus respectivos fabricantes e detentores de direitos autorais (como **Mercedes-Benz**, **Marcopolo**, **Thermo King**, etc.). Os créditos estão devidamente atribuídos a cada fornecedor nos documentos e seções correspondentes. O TecVander apenas disponibiliza e organiza o acesso a estes materiais.
